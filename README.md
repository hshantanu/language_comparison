# Language Comparison
Compare Languages across multiple aspects such as ->
> JSON parsing
> DB operations
> Concurrency
> Regex Matching
> Socket programming
> Inmemory cache
> Image processing

Elixir launch REPL -
```sh
$ iex
```
Elixir Execute program
```sh
$ elixirc hello_world.exs
```
Elixir Mix fetch dependencies
```sh
$ mix deps.get
```
Elixir Execute program using mix
```sh
$ mix run hello_world.exs
```
